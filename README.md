## 🚀 Sobre el proyecto

Este proyecto simula un sistema operativo de escritorio ("Pinboard OS") para presentar mi trabajo como Director de Arte. Diseñado para ofrecer una experiencia inmersiva que rompe con el scroll tradicional.

### 🛠️ Ficha Técnica
- **Lenguajes:** HTML5, CSS3, JavaScript (Vanilla - Sin Frameworks).
- **Estilo:** Tailwind CSS + Bootstrap (Grid).
- **Diseño:** Responsive (adaptable a móvil y escritorio).
- **Concepto:** Interfaz de ventanas arrastrables, sistema de archivos y narrativa visual.

## 📂 Contenido del Portafolio
1.  **Dirección de Arte & Branding:** Identidades visuales y logotipos.
2.  **Campañas 360º:** Estrategia y ejecución publicitaria.
3.  **Diseño UX/UI:** Prototipado y diseño de aplicaciones.
4.  **Social Media:** Estrategias de crecimiento orgánico y viralidad.
